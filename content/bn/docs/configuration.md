# কনফিগারেশন বিকল্পসমূহ

আপনার এজেন্সির হোয়াইট-লেবেল থিমগুলি কাস্টমাইজ করুন।

## টোকেন ফাইলসমূহ

টোকেনগুলি `/styles/tokens` এ সংজ্ঞায়িত করা রয়েছে।

## ব্র্যান্ড এট্রিবিউট

`html` ট্যাগে `data-brand` এট্রিবিউট সেট করুন।
